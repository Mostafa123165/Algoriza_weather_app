# Weather Application
Weather is a mobile application developed using Flutter. It supports both Android and IOS.

## Folder structure

```
Weather-App
├── android
├── assets
├── build
├── ios
├── lib
└── test
```

```
lib
├── BlockOpserver
├── components
├── home
├── network
├── share
└── main.dart
```

```
home
├── Block
├── home_Screen
```

```
model
├── weather-date
```

```
network
├── local
├── dio_helper
```

```
share
├── style 
├── themes 
├── Colors
```
